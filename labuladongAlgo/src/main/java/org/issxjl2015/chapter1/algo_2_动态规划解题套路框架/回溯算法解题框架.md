# 回溯算法解题框架

解决回溯问题，实际上就是一个决策树的遍历过程。
>1、路径：也就是已经做出的选择
>
>2、选择列表：也就是你当前可以做的选择
>
>3、结束条件：也就是到达决策树底层，无法再做选择的条件

回溯算法的框架：
```java
int[] result = new int[n];
public int[] backtrack(路径， 选择列表)：
    if (满足结束条件) {
        result.add( 路径 );    
        return;
    }
    for (int 选择 ： 选择列表) {
        做选择
        backtrack(路径， 选择列表)
        撤销选择
    }
```

其算法的核心就是 for 循环里面的递归，在递归调用之前 “做选择”，在递归调用之后 “撤销选择”。
