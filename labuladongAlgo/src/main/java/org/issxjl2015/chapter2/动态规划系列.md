## 2.1、动态规划设计：最长递增子序列
最长递增子序列（Longest Increasing Subsequence, 简写 LIS）是一个非常经典的算法问题，   
题目如下：
> 给你一个整数数组 nums ，找到其中最长严格递增子序列的长度。  
> 子序列是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。  
> 例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的子序列。  
> 函数为： public int lengthOfLIS(int[] nums)
